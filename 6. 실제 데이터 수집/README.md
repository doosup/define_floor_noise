## 6. 실제 데이터 수집

기숙사 장소를 대여하여 실제 데이터를 수집하였다. 같은 위치의 5층과 6층 장소를 대여하여 아래 6가지 행위에 대한 데이터를 수집하였다.

1. walking: 걷는 행위

2. random walking: 무작위 강도로 걷는 행위

3. running: 뛰는 행위

4. hammer: 망치로 벽을 치는 행위

5. rolling hammer: 망치와 아령과 같은 물체가 바닥에 끌리는 행위

6. table: 탁자를 바닥에 끄는 행위
